1.2.2 气体摩尔体积

大本（P13-15）

典例1：B

  1. 0.5mol O2含有的分子数为0.5NA，11.2L O2因为没有说是标准状况，所以无法计算其物质的量。如果有说是标准状况，那么\(V_{\mathrm{m}}=22.4\ \mathrm{L/mol}\)，根据\(V_{\mathrm{m}}=\frac{V}{n}\)便可以算出O2的物质的量，进而算出分子数。
  2. 常温常压在这里是个无关条件，根据\(n=\frac{m}{M}\)，18g H2O 的物质的量\(n=\frac{m}{M}=\frac{18\ \mathrm{g}}{18\ \mathrm{g/mol}}=1\ \mathrm{mol}\) 。而1mol H2O有2mol H和1mol O，共3mol原子，所以含有的原子总数为3mol。
  3. 因为CCl4（四氯化碳）是液体，因此不能用\(V_{\mathrm{m}}=\frac{V}{n}\)来计算体积，所以该选项错误。
  4. 常温常压下，无法使用\(V_{\mathrm{m}}=\frac{V}{n}\)来计算体积，所以该选项错误。

训练1：B

  1. 没有说是标准状况，所以该选项错误。如果有说是标准状况，则正确。
  2. 标准状况下，0.2mol H2的体积V(H2)=Vm·n(H2)=22.4L/mol×0.2mol，0.8mol CO2的体积V(CO2)=Vm·n(CO2)=22.4L/mol×0.8mol，两者加起来就是22.4L。
  3. 应把「物质」改为「气体」才对。
  4. 没有说标准状况，无法通过体积计算CO2的物质的量。

典例2：D

由容积相同的容器充满气体这个条件可知，O2和O3的体积V相等。

  1. 根据\(pV=nRT\)或\(V_{\mathrm{m}}=\frac{V}{n}\)可知，同温同压下，相同体积的气体的物质的量也相等。
  2. 刚刚选项A已经得出它们的物质的量n相等，所以根据\(m=n\cdot M\)可计算氧气和臭氧的质量，其中氧气的摩尔质量M(O2)=32g/mol，臭氧的摩尔质量M(O3)=48g/mol，因此计算出来的质量不相等。
  3. 根据选项A可知它们物质的量相等，所以假设都是1mol，此时O2含有的原子1mol×2=2mol，O3含有的原子1mol×3=3mol。
  4. 同理假设它们的物质的量均为1mol，此时它们均有1mol分子，故D选项正确。

训练2：B

  1. 「物质」应改为「气体」。
  2. 由物质的量推算出分子数，选项说是等物质的量，那么假设均有1mol，可得1mol乙烯的有1mol乙烯分子，1mol一氧化碳有1mol一氧化碳分子，故分子数相等，且不受外界条件影响。
  3. 根据\(V_{\mathrm{m}}=\frac{V}{n}\)，质量\(m=n\cdot M=\frac{V}{V_{\mathrm{m}}}\cdot M\)，但此时没有说明温度和压强，所以Vm的值不确定，答案是不一定。
  4. 「物质」应改为「气体」。

训练3：D

  1. 略
  2. 略
  3. 含有碳元素，属于有机物。
  4. 碳原子数较多，在标准状况下不可能是气体。

1：C

  1. 水在标准状况下是液体，不能使用\(V_{\mathrm{m}}=\frac{V}{n}\)计算体积，只能先计算1mol水的质量，\(m=n\cdot M=1\mathrm{mol}\cdot 18\mathrm{g/mol}=18\mathrm{g}\)，再根据水的密度为1g/mL可计算得其体积为18mL。
  2. 应该是在标准状况下。
  3. 选项已说明是在标准状况下，所以可知Vm=22.4L/mol，由混合气体总共有1mol计算得总体积为22.4L。
  4. 应为「标准状况」而非「通常状况」。

2：D

阿伏伽德罗定律为：同温同压下，相同体积的任何气体具有相同的体积。其中A选项和B选项是根据阿伏伽德罗定律得出来的结论，而非其本质原因。C选项表述错误，不同的气体分子大小不同，因为组成气体分子的原子的大小是不同的。D选项正确，因为平均距离相等，再加上气体分子之间距离大到可以忽略气体分子的大小，进而把气体分子看作质点，所以是阿伏伽德罗定律能够成立的本质原因。

3：D

  1. 四氯化碳在标准状况下是液体，而非气体。
  2. 标准状况下，空气是混合气体，但其体积仅与分子数有关，所以标准状况下22.4L空气含有NA个气体分子，但选项说的是单质分子，空气中除了有氧气氮气等单质分子之外，还有二氧化碳等化合物分子。
  3. 「常温常压」应改为「标准状况」。
  4. \(n=\frac{m}{M}=\frac{23\mathrm{g}}{46\mathrm{g/mol}}=0.5\mathrm{mol}\)，1mol二氧化氮有2mol氧原子，现在有0.5mol二氧化氮，所以有1mol氧原子，即含有NA个氧原子。

4：C

  1. 根据\(pV=nRT\)，同温同压下，体积之比等于物质的量之比，与摩尔质量无关。例如标准状况下即满足同温同压，1mol氢气和1mol氧气的体积均为22.4L，体积比为1:1，等于物质的量之比1:1，但不等于摩尔质量之比2:32。
  2. 根据\(pV=nRT\)，同温同压下，物质的量之比等于体积之比，选项问的是密度之比，把公式转换为\(p\frac{m}{\rho}=nRT\)，再把质量用摩尔质量和物质的量表示，得\(p\frac{n\cdot M}{\rho}=nRT\)，因此两种气体的物质的量之比不等于密度之比。
  3. 由选项B推出的公式\(p\frac{n\cdot M}{\rho}=nRT\)约掉等号两侧的n，可得\(p\frac{M}{\rho}=RT\)，因此气体摩尔质量之比等于密度之比。
  4. 根据\(pV=nRT\)和\(n=\frac{m}{M}\)得\(pV=\frac{m}{M}RT\)，此时还与两种气体的摩尔质量有关。

5：B

  1. 氧气的摩尔质量是32g/mol，乙烯（C2H4）的摩尔质量是28g/mol，当两者等物质的量时，质量并不相等。当有28mol氧气和32mol乙烯时，他们的质量相等，即28mol×32g/mol=32mol×28g/mol。此时氧气的分子数是28NA，乙烯的分子数是32NA，并不相等。
  2. \(n=\frac{m}{M}=\frac{\rho V}{M}\)，一氧化碳和乙烯的摩尔质量均为28g/mol，所以n相等，即分子数相等。
  3. 根据理想气体状态方程\(pV=nRT\)，氧气和氮气同温同体积，但是压强未知，所以物质的量不一定相等，即分子数不一定相等。
  4. 与C选项同理，温度未知，物质的量不一定相等。

6

  1. 17g氨气即为1mol NH3，含有1mol氨气分子，而1mol O2在标准状况下的体积即为22.4L，氧原子数为2mol。
  2. 先根据气体在标准状况下的体积算出其物质的量，\(n=\frac{V}{V_{\mathrm{m}}}=\frac{2.24\mathrm{L}}{22.4\mathrm{L/mol}}=0.1\mathrm{mol}\)，再根据定义计算摩尔质量\(M=\frac{m}{n}=\frac{3.65\mathrm{g}}{0.1\mathrm{mol}}=36.5\mathrm{g/mol}\)
  3. 标准状况下22.4L二氧化碳即为1mol，含有的氧原子2mol，即要求水中氧原子的数目也是2mol。已知1mol H2O中氧原子是1mol，所以要使氧原子为2mol，则需要2mol H2O，即36g。

小本（P95）

1：B

物质的体积大小由粒子的数目、大小和距离所决定，而气体粒子之间距离大到可以忽略粒子本身的大小，且不同气体粒子在同温同压下的距离是相等的，所以气体的体积在同温同压下仅由粒子的数目所决定，即由气体物质的量所决定。此外也可通过理想气体状态方程\(pV=nRT\)推出这个结论。

2：D

\(V=V_{\mathrm{m}}\cdot n=V_{\mathrm{m}}\cdot \frac{m}{M}\)，可知摩尔质量越大，体积越小。

3：B

  1. 气体体积与气体分子大小无关，同温同压下，相同物质的量的气体具有相同的体积。
  2. 根据\(pV=nRT\)可知B选项正确。
  3. 定义错误，不一定是22.4L/mol。
  4. 根据\(pV=nRT\)可知气体体积还与温度和压强这两个外界条件有关，所以不一定。

4：C

  1. 未说明是标准状况，不一定是22.4L。
  2. 在标准状况下，即在0℃和101kPa下是22.4L/mol，但根据\(pV=nRT\)可知当压强和温度同时变化时也可能出现22.4L/mol的情况。
  3. 在计算气体时，不同气体分子之间的差异可以忽略不计，因此也适用于混合气体。
  4. 未说明是标准状况，如果说明是标准状况则正确。

5：B

12g氢气则是6mol，根据同温同压下，相同体积的气体含有相同的粒子数，所以氮气也是6mol，质量\(m=n\cdot M=\mathrm{6mol\cdot 28g/mol=168g}\)，体积\(V=V_{\mathrm{m}}\cdot n=\mathrm{22.4L/mol\cdot 6mol=134.4L}\)。

6：C

  1. 未指明是标准状况，无法计算氮气的分子数。
  2. 计算分子数即是计算物质的量，根据\(n=\frac{m}{M}\)，此时质量相同均为5g，但一氧化碳和氢气的摩尔质量不同，所以分子数不同。
  3. 1mol H2O的氢原子为2mol，现在氢原子有1mol，所以可以计算出H2O为0.5mol。
  4. 假设氨气和氯化氢均为1mol，此时氨气有10mol质子，氯化氢有20mol质子，要使质子数相等，则氨气和氯化氢的物质的量之比应为2:1，所以分子数不同。

7：D

  1. 未指明是标准状况，无法计算。
  2. 未指明标准状况，无法计算。
  3. 水在标准状况下不是气体。22.4L水的质量应该通过密度来计算，得其质量为22.4kg。
  4. 22g二氧化碳的物质的量是0.5mol，标准状况下11.2L氯化氢的物质的量是0.5mol，所以分子数相同，均为0.5NA。

8：A

氢气（H2）是双原子分子，氦气（He）是稀有气体，是单原子分子。所以空心双球是氢气，实心单球是氦气，排除B选项。此外题目要求等质量，根据\(m=n\cdot M\)，而氢气的摩尔质量为2g/mol，氦气的摩尔质量为4g/mol，所以氢气和氦气的物质的量之比应为2:1才能让质量相等，所以排除物质的量之比是1:1的D选项。又根据\(V=V_{\mathrm{m}}\cdot n\)，所以氢气和氦气的体积比应为2:1，选项A正确。

9

  1. 根据阿伏伽德罗定律，同温同压下，相同物质的量的气体具有相同的体积，即\(V_{\mathrm{m}}=\frac{V}{n}\)，Vm为定值，分子数相等，所以体积相等，即容积比为1:1。
  2. 等物质的量的氧气和臭氧的原子数之比是2:3，要使原子数相等，则需要使氧气和臭氧的物质的量之比为3:2，此时原子数之比是2×3:3×2，即原子数相等，此时根据阿伏伽德罗定律，同温同压下气体物质的量之比等于容积（体积）之比，所以是3:2。
  3. 根据\(V_{\mathrm{m}}=\frac{V}{n}\)，同温同压下体积比3:2，所以物质的量之比也是3:2。质量之比通过\(m=n\cdot M\)计算，即3mol×32g/mol : 2mol×48g/mol=1:1。密度根据\(\rho=\frac{m}{V}\)计算，\(\frac{1}{3}:\frac{1}{2}=2:3\)。
